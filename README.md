<div align=left><h1> 부기대여 </h1></div>
AI를 활용한 기자재 대여 시스템
<br><br><br>


### 🖼️ 프로젝트 소개
비전문가도 기자재를 쉽게 파악할 수 있게 하고, 체계적으로 관리할 수 있게 돕는 어플 입니다.
학생들도 사용자로 어플을 이용할 수 있으며 대여 및 반납을 쉽게 할 수 있습니다.
<br>
<br>


### 📅 개발기간
* 24.3 ~ 24.5
<br>

### 💡 주요 기능
* 공통 기능 : 기자재 인식 및 분류
  
| 사용자 | 관리자 |
| ----------- | ------------- |
| 기자재 대여 및 반납 | 기자재 대여 및 반납 관리 |
| 반납 시간 알림 받기 |  |
|  | 블랙리스트 사용자 설정 |
<br>
